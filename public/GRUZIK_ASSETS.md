# Аватар Грузика — анимация

Чтобы в чате отображалась **анимация** Грузика (GIF), в эту папку нужно положить:

- **gruzik.gif** — анимированный GIF (основной вариант)
- при желании **gruzik.webm** — видео-анимация (fallback, если GIF не загрузится)

Сейчас в папке есть только **gruzik.png** и **gruzik.jpg** — это статичные картинки, поэтому анимация не воспроизводится: код сначала запрашивает `gruzik.gif` → получает 404 → переходит на `gruzik.webm` → 404 → показывает `gruzik.png`.

**Что сделать:** добавьте файл `gruzik.gif` (вашу анимацию Грузика) в папку `public/`. После пересборки приложения аватар начнёт показывать анимацию.

Дополнительные варианты по эмоциям: `gruzik-typing.gif`, `gruzik-happy.gif` и т.д. (см. код `GruzikAvatar` в App.tsx).
