## 2. Навыки, которые стоит явно прописать (для Алисы и для клиентов)

Ниже — формулировки «навыков» в формате: что умеет система / что может клиент. Их можно вставить в описание навыка Алисы, в раздел «Возможности» на сайте или в памятку для клиента.

### 2.1 Голос (Алиса)

- Узнать **перевозки в пути** — «Какие перевозки в пути?», «Что в пути?».
- Узнать **счета на оплату** — «Что требует оплаты?», «Какие счета на оплату?».
- Получить **подробности** после краткого ответа — «Подробнее», «Да».
- **Отвязать компанию** — «Отвяжи компанию» (если реализовано в навыке).
- **Авторизация** — один раз ввести код из мини‑приложения; дальше Алиса работает от имени выбранной компании.
- **Контакты и общие вопросы** — «Контакты Холз», «Статус перевозки номер X» — ответ через AI‑чат.

### 2.2 Мини‑приложение

- Смотреть **сводку** по перевозкам и суммам на дашборде.
- Фильтровать перевозки по **статусу**, **типу**, **периоду**, **отправителю/получателю**.
- Отслеживать **SLA** и сроки доставки.
- **Скачивать документы** (ЭР, Счёт, УПД, АПП) по короткой ссылке из карточки груза.
- Вести **несколько компаний** в одном аккаунте и переключаться между ними.
- Задавать вопросы в **AI‑чате** с контекстом по активной компании; при необходимости отвязать компанию в чате.
- Включить **2FA** (Google Authenticator или Telegram).
- Добавлять перевозки в **избранное**.

### 2.3 Telegram

- Спрашивать про перевозки и документы в боте.
- Получать ссылки на документы по перевозке.

## 4. Готовые формулировки для карточки навыка Алисы

**Краткое описание (для каталога):**  
«Узнавайте голосом, какие перевозки в пути и какие счета ждут оплаты. Навык работает от имени вашей компании в Холз после ввода кода из мини‑приложения.»

**Что умеет навык:**  
- Перевозки в пути — список и подробности по запросу.  
- Счета на оплату — список и подробности.  
- Контакты Холз и общие вопросы — через голосовой чат.  
- Безопасная привязка к компании по коду из мини‑приложения.

**Как начать:**  
Скажите: «Запусти навык Холз». Получите код в мини‑приложении HAULZ (Профиль → Голосовые помощники → Алиса), назовите код Алисе — после этого можно спрашивать про перевозки и оплаты голосом.
