# Виджет Bitrix24 «Открытая линия» — только на странице «Чат»

Чтобы виджет открытой линии показывался **только на странице «Чат»** (вкладка «Поддержка»), в настройках виджета Bitrix24 укажите ограничение по URL.

## Ссылка страницы чата

Страница чата в приложении открывается с параметром `tab=support`.

**Полная ссылка (подставьте свой домен):**
```
https://ВАШ_ДОМЕН/?tab=support
```

Если приложение развёрнуто не в корне, например на `https://example.com/mini/`:
```
https://ВАШ_ДОМЕН/ПУТЬ/?tab=support
```

**Универсальный шаблон для ограничения в Bitrix24** (виджет показывать только при наличии `tab=support` в URL):
```
*tab=support*
```

## Где прописать ограничения в Bitrix24

1. **Битрикс24** → Настройки → Открытые линии → Ваша линия (или создайте виджет).
2. В настройках виджета найдите блок **«Ограничение по страницам»** / **«Показывать на страницах»** / **«Разрешённые адреса»**.
3. Укажите один из вариантов:
   - точный URL: `https://ваш-домен.ru/?tab=support`;
   - или маску: `*tab=support*` — виджет будет работать только когда в адресе есть `tab=support` (т.е. только на странице «Чат»).

Так виджет не будет открываться на других вкладках приложения (Грузы, Документы, Профиль и т.д.).
